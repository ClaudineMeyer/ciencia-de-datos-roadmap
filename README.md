## Funciones y Aplicaciones
- ETL, EDA, Data Viz y ML
- Prediccion entre el 65% y 70% de fiabilidad de la enfermedad de Alzheimer
  - Datos de entrada: edad, nivel cognitivo y presencia de gen APOE4
